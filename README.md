# Pulsey
Pulsey is a small utility for switching between different sound modules (cards, bluetooth speakers, headphones) and sound profiles easily in terminal. It is a wrapper on top of `pacmd` that heavy uses `dialog`.

## Dependencies
- dialog
- pacmd
